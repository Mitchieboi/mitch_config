For now, I will use LazyVim as my neovim distro.

[Link to LazyVim](https://www.lazyvim.org/)

## Steps to Install
1. Install the Hack NerdFont from [here](https://www.nerdfonts.com/font-downloads)
2. Ensure curl, a C compiler, and git are installed
3. Follow the instructions [here](https://www.lazyvim.org/installation) to install LazyVim
